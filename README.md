Sorteador de personagens de one piece.

O projeto foi desenvolvido utilizando HTML, CSS e JavaScript, consumindo uma API pública para obtenção dos personagens do anime One Piece.

O deploy da aplicação foi realizado utilizando o GitHub Pages, permitindo acesso público via navegador.


O projeto foi aprimorado com foco em experiência mobile, utilizando design responsivo e melhorias visuais.

Foram avaliadas as métricas utilizando o Lighthouse, garantindo boas práticas de performance, acessibilidade e SEO.

A aplicação foi transformada em um PWA (Progressive Web App), permitindo instalação no dispositivo e funcionamento aprimorado.

Também foi implementado o uso de hardware através da API de vibração, aumentando a interatividade do usuário.
